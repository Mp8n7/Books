# Suno Prompts

This repository is a collection of prompts for Suno AI, a generative audio model.

## Features

- **Advanced Prompting Guide**: `prompting_guide.md` contains expert strategies for realism, genre control, and metatags.
- **Prompt Examples**: `examples.md` for quick inspiration.
- **Utilities**: `scripts/count_prompt.py` to ensure your prompts fit within Suno's 1000-character limit.

## Structure

- `prompts/`: (Optional) Directory for storing your saved prompt files.
- `scripts/`: Helper scripts.

## How to use

1. Read the [Prompting Guide](prompting_guide.md) to master Suno's prompt engineering.
2. Use `python scripts/count_prompt.py "your prompt"` to check length.
3. Generate music!
