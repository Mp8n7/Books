# Suno Prompts

This repository is a collection of prompts for Suno AI, a generative audio model.

## Structure

- `prompts/`: Contains the prompt files, organized by genre or type.
- `examples.md`: Examples of effective prompts.
- `prompting_guide.md`: A detailed guide on how to structure prompts and use metatags.

## How to use

1. Read the [Prompting Guide](prompting_guide.md) to understand how to craft effective prompts.
2. Check [examples.md](examples.md) for inspiration.
3. Copy the prompts and use them in Suno to generate music.
